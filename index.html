What is HouseBite?

A web app that turns your household into a shared smart pantry — helping you waste less, spend sless and keep track of what you already have.

HouseBite gives households a shared space to manage food more effectively. Create a household or join an existing one, add products with expiry dates, see what is about to expire, and get recipe suggestions based on what you already have at home.

Beyond the pantry, HouseBite helps you stay on top of grocery spending with a shared monthly budget, lets members scan receipts to log groceries purchases automatically, and supports individual dietary restrictions so meal planning works for everyone.

Why it matters

53% of food waste in the EU comes from households
69 kg of food waste per person is generated by EU households each year

Source: European Commission, based on Eurostat 2025.

Development progress

Sprint overview

Sprint 1 Completed

Project Foundation

Core project setup, authentication, and the shared household model that the rest of the app builds on.

  • Project setup — Vite + React + TypeScript, Docker dev environment
  • Supabase Auth with automatic family_member trigger on sign-up
  • Households: create, join via invite code, member allocations
  • Database schema across 12 tables with Row Level Security
Sprint 2 Completed

Core Product Experience

Focused on delivering the core user-facing features that define the everyday HouseBite experience.

  • Household dashboard with a filterable “expiring soon” view
  • Recipe suggestions powered by Spoonacular, with support for diets and intolerances
  • Receipt scanning
  • Account page with dietary restriction profiles for each member

Under the hood

Tech & architecture

HouseBite is a progressive web app — installable on phones and desktops, with a Supabase backend and on-device processing where it matters.

Frontend

  • React 19 + TypeScript
  • Vite + React Compiler
  • Mantine UI + Tabler Icons
  • React Router 7
  • PWA manifest + icons

Backend & data

  • Supabase Auth
  • PostgreSQL with Row Level Security
  • Edge Functions (Deno)
  • SECURITY DEFINER RPCs
  • 12 tables, 1 derived view

Integrations

  • Spoonacular — recipe search
  • Tesseract.js — on-device OCR
  • Open Food Facts — barcode data

Workflow

  • Scrum methodology
  • Sprint-based delivery
  • KTH GitLab + GitHub dual remote
  • Docker for local dev
